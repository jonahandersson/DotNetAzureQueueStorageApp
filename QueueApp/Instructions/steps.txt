1. Azure CLI - Create Storage account using this  command 

az storage account create 
--name articlesjcadev 
-g learn-64b65e20-e800-4a3a-859e-9030044d7eab 
--kind StorageV2 
--sku Standard_LRS

-- To Show connection String 
az storage account show-connection-string --name articlesjcadev --resource-group learn-64b65e20-e800-4a3a-859e-9030044d7eab
